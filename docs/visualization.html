Visualization¶

Text visualization

texthero.visualization.scatterplot(df: pandas.core.frame.DataFrame, column: str, color: str, hover_data: ]) → None¶

Scatterplot of df[column].

The df[column] must be a tuple of 2d-coordinates.

Usage example:

```

import texthero

df = pd.DataFrame([(0,1), (1,0)], columns=’pca’) texthero.visualization.scatterplot(df, ‘pca’)

```
